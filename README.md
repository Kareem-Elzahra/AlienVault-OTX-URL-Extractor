# AlienVault-OTX-URL-Extractor
chrome extention to fetch url from AlienVault OTX
How to Install and Use This Extension:
Create the folder structure with all 4 files above

Load the extension in Chrome:

Go to chrome://extensions/

Enable Developer mode

Click "Load unpacked"

Select your alienvault-json-fetcher folder
Use the extension:

Navigate to: https://otx.alienvault.com/api/v1/indicators/domain/target.com/url_list?limit=1000&page=1

Click the extension icon in your toolbar

Click "Extract URLs from JSON"
